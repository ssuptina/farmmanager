# 🐔 Farm Manager

A full-stack poultry farm management system designed to help farmers efficiently manage daily operations, including flock management, inventory, sales, expenses, worker management, and business analytics through a modern web interface.

## 🌟 Features

- 📊 Interactive dashboard with farm analytics
- 🐔 Flock and health management
- 📦 Inventory and stock tracking
- 💰 Sales, expenses, and invoice management
- 👥 Customer and worker management
- 📅 Task scheduling and vaccination tracking
- 📈 Reports and data visualization
- 📱 Responsive design for desktop and mobile

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/ssuptina/farmmanager.git
cd farmmanager
```

### Backend

```bash
cd backend
npm install
npm run dev
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

The application will run locally after starting both servers.
